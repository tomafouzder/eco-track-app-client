# Eco Track Web Application

## Project Name
**Eco Track Website**
**Eco Track App**

## Purpose
Eco Track is a responsive and interactive sustainability-focused web application designed to encourage eco-friendly habits and community engagement. The platform allows users to explore environmental challenges, join activities, share eco-tips, and track personal progress. It aims to promote greener lifestyles through real-time updates, dynamic visuals, and community-driven motivation. Eco Track helps users reduce waste, save energy, improve health, and contribute to a cleaner and healthier planet.. ---->

## Key Features
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices.
- **Challenge Listing & Details**: Users can explore environment-related challenges with full descriptions, icons, and images.
- **Join Challenge System**:Users can join challenges, track status, and update progress with toast notifications for feedback.
- **Hero Slider**: Engaging **Swiper slider** with animated text overlays using **Framer Motion** and **Animate.css**.
- **Eco Tips Section**:Users can view and share environment-friendly tips.
- **Firebase Integration**: For authentication, data management, or real-time updates.
- **Interactive Animations**: Smooth animations on sliders, buttons, and page elements using **Animate.css** and **Framer Motion**.
- **Toast Notifications**: Instant feedback for joining and form interactions using **react-hot-toast** and **sweetalert2**.
- **Event & Progress Tracking**: Displays community events and user activity in real time.
